### Sviluppato per linux

## Per installare le dipendenze

`pip3 install --user -r requirements.txt`

## Per utilizzarlo via linea di comando

`cd cli-ui`

`python3 client.py`

## Per utilizzarlo via web

`cd web-ui`

`python3 web-ui.py`

## Per vedere le credenziali

`cd tools`

`python3 view-cred.py`
